# Install
- copy the cloned in `.rmp/plugins` folder

# Configurations
```lua
{
    "gruvbox-pine-rmp",
    -- default config
    {
        VARIANT = "dark", --"dark" "light"
        name_as = "gruvbox"
    }
}
```

- sins the plugin is activated u have a new theme available "gruvbox" by default or your alias that u named in plugin configuration
